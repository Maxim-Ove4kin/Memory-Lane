# Active Context

## Текущий фокус
Репозиторий приведён к чистой структуре обычного Vite-проекта без legacy-хвостов от старого стека.

## Активные задачи
- Cleanup завершён: репозиторий переведён на единый актуальный стек `Vite + Vanilla JS + bun`
- Tooling приведён к обычному состоянию проекта: добавлены scripts для `Biome`, удалены неиспользуемые файлы из корня
- Следующий фокус: разработка продуктовых фич внутри уже очищенной структуры проекта

## Принятые решения
- Технология: Vanilla JS + Vite (без фреймворков)
- PWA-подход для мобильной работы
- Минималистичный UI с фокусом на контенте
- В репозитории должен остаться один актуальный набор документов без дублирующих PRD-файлов

## Известные неопределённости
- Структура данных для хранения воспоминаний (localStorage / IndexedDB)
- Реализация AI-редактора (внешнее API или локальная эмуляция)
- Карта: Google Maps API или OpenStreetMap
