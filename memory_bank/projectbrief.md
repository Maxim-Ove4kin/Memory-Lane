# Project Brief

## Goals
Recipe Finder is a web application for discovering recipes by dish name or by available ingredients. The product targets a young audience and emphasizes an expressive hand-drawn interface with responsive behavior.

## Scope
The confirmed project scope comes from `PRD.md`, `docs/development_plan.md`, and the current codebase. The implemented scope today is limited to the initial frontend shell with homepage and mocked search results.
