# Установка Memory Lane как приложения

## Перед установкой

Для локальной проверки запустите проект через Vite:

```bash
bun install
bun run dev
```

Для полноценной установки как PWA приложение должно быть доступно по `https` или через `localhost`.

## iOS

1. Откройте приложение в Safari.
2. Нажмите кнопку «Поделиться».
3. Выберите «На экран Домой».
4. Подтвердите добавление.

## Android

1. Откройте приложение в Chrome.
2. Откройте меню браузера.
3. Выберите «Добавить на главный экран» или «Установить приложение».
4. Подтвердите установку.

## Desktop

1. Откройте приложение в Chrome или Edge.
2. Нажмите иконку установки в адресной строке.
3. Подтвердите установку.

## Ограничения текущей версии

- данные сохраняются локально в браузере;
- полноценный офлайн-режим ещё не завершён;
- service worker пока не реализован.
