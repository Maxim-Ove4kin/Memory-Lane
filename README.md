# Memory Lane

Memory Lane — мобильное PWA-приложение для сохранения истории закрытых групп: друзей, семьи и команд.

## Что уже есть

- главный экран со списком групп и поиском;
- экран ленты событий внутри группы;
- экран участников;
- экран карты памяти;
- экран ачивок;
- экспорт и импорт данных через JSON;
- светлая и тёмная тема;
- локальное хранение данных в браузере.

## Текущий стек

- `Vite`
- `Vanilla JavaScript`
- `HTML5`
- `CSS3`
- `localStorage`
- `Leaflet` для карты

## Запуск

```bash

```

Сборка:

```bash
bun run build
```

Форматирование и базовая проверка:

```bash
bun run format

```
